# 公众号写作系统

> 从选题到复盘的全流程公众号内容创作系统

## 项目简介

本仓库是一套完整的公众号内容创作方法论和操作手册，涵盖选题策划、内容写作、排版设计、发布运营、数据分析五个核心环节。每个环节都有详细的SOP文档和实操模板，帮助内容创作者系统化提升创作效率和质量。

## 核心功能

### 一、选题策划

科学的选题是爆款的第一步。我们提供：

- **选题方法论**：从用户需求、热点趋势、竞品分析三个维度寻找选题
- **热点追踪SOP**：如何快速捕捉热点并进行内容创作
- **选题评估表**：用打分卡量化评估选题潜力

### 二、内容写作

优质内容的核心是结构和表达。我们提供：

- **标题公式库**：50+经过验证的标题公式，提升打开率
- **结构模板库**：10种常见文章结构，快速搭建内容框架
- **金句创作法**：如何写出令人印象深刻的核心观点

### 三、排版设计

好的排版能大幅提升阅读体验。我们提供：

- **排版规范**：字号、行距、配色、间距的标准化建议
- **排版模板**：可直接使用的HTML模板，支持微信公众号编辑器

### 四、发布运营

发布时机和方式影响传播效果。我们提供：

- **发布检查清单**：发布前的最后检查
- **最佳发布时间**：不同类型内容的发布时机建议

### 五、数据复盘

数据驱动内容迭代。我们提供：

- **数据分析方法**：关键指标解读和优化方向
- **内容优化SOP**：如何根据数据反馈优化内容

## 快速开始

### 1. 选题阶段

```bash
# 查看选题方法论
open 选题/选题方法论.md

# 使用选题评估表
open 选题/选题评估表.md
```

### 2. 写作阶段

```bash
# 查找合适的标题公式
open 写作/标题公式库.md

# 选择文章结构
open 写作/结构模板库.md
```

### 3. 排版阶段

```bash
# 使用排版模板
open 排版/排版模板.html

# 参考排版规范
open 排版/排版规范.md
```

### 4. 发布阶段

```bash
# 发布前检查
open 发布/发布检查清单.md
```

### 5. 复盘阶段

```bash
# 数据分析
open 复盘/数据分析方法.md
```

## 使用建议

1. **新人入门**：按照选题→写作→排版→发布→复盘的顺序逐个学习
2. **日常创作**：使用对应的SOP和模板提高效率
3. **持续优化**：根据数据复盘结果调整内容策略
