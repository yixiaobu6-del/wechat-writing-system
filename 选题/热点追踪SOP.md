# 热点追踪SOP

> 如何快速捕捉热点并进行内容创作

## 一、热点发现机制

### 1. 建立热点监控矩阵

| 渠道 | 监控频率 | 关注点 |
|------|----------|--------|
| 微博热搜 | 每小时 | 全网热点 |
| 抖音热点榜 | 每小时 | 年轻用户关注 |
| 知乎热榜 | 每天 | 深度话题 |
| 百度指数 | 每天 | 搜索趋势 |
| 微信指数 | 每天 | 微信生态热度 |

### 2. 热点筛选标准

- **时效性**：热点发生后24小时内最佳
- **相关性**：与账号定位的关联程度
- **争议性**：是否能引发讨论
- **情感性**：是否能引发共鸣
- **安全性**：是否涉及敏感话题

## 二、热点响应流程

### Phase 1: 快速判断（15分钟内）

1. **热点核实**：确认热点真实性，避免翻车
2. **风险评估**：是否有政治、法律、道德风险
3. **关联判断**：是否能与账号定位结合
4. **竞争分析**：是否已有大量竞品内容

### Phase 2: 内容规划（30分钟内）

1. **确定角度**：找到差异化切入点
2. **确定形式**：图文/视频/条漫
3. **确定节奏**：什么时候发布最有利
4. **准备素材**：收集相关资料和图片

### Phase 3: 快速创作（2小时内）

1. **框架搭建**：快速搭建文章骨架
2. **核心观点**：形成1-2个核心观点
3. **素材填充**：填充案例和数据
4. **标题优化**：打磨吸引人的标题

### Phase 4: 质量检查（30分钟内）

1. **事实核查**：数据和事实是否准确
2. **风险自查**：是否存在敏感表述
3. **排版优化**：提升阅读体验
4. **预发布检查**：封面图、摘要

## 三、热点切入角度

### 角度一：解读分析
深入分析热点背后的原因、影响和趋势

### 角度二：借势科普
借用热点科普相关知识

### 角度三：情感共鸣
从热点引发的情感角度切入

### 角度四：争议讨论
提出不同观点引发讨论

### 角度五：实用指南
提供应对热点的实用建议

## 四、热点创作模板

### 模板一：热点解读型
```
开头：热点事件回顾（100字）
分析：事件背后的原因（300字）
延伸：同类事件对比（200字）
观点：我的核心看法（200字）
结尾：引发讨论的问题（100字）
```

### 模板二：借势科普型
```
开头：热点引入（50字）
科普：核心知识讲解（500字）
应用：如何运用（300字）
结尾：总结要点（100字）
```

### 模板三：情感共鸣型
```
开头：热点场景描写（100字）
共鸣：读者可能的感受（300字）
故事：1-2个具体案例（400字）
升华：价值观点（200字）
结尾：互动问题（50字）
```

## 五、热点避坑指南

### 必须避免
1. 未经核实的信息
2. 涉及政治敏感话题
3. 可能引发法律纠纷的内容
4. 对逝者不尊重的表达
5. 歧视性言论

### 需要谨慎
1. 涉及商业利益的话题
2. 可能侵犯肖像权的内容
3. 引用他人观点时的版权
4. 热点反转后的风险
