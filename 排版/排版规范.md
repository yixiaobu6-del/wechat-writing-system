# 排版规范

> 公众号文章排版标准化指南

## 一、字体规范

### 正文字体

| 项目 | 推荐值 | 说明 |
|------|--------|------|
| 字号 | 15-17px | 最佳阅读体验 |
| 颜色 | #333333 或 #3e3e3e | 深灰色，不刺眼 |
| 行距 | 1.5-1.75 | 适当留白 |

### 标题字体

| 级别 | 字号 | 加粗 | 说明 |
|------|------|------|------|
| 一级标题 | 18-20px | 是 | 文章主标题 |
| 二级标题 | 16-18px | 是 | 章节标题 |
| 三级标题 | 15-16px | 是 | 小节标题 |

### 引用文字

| 项目 | 推荐值 |
|------|--------|
| 字号 | 14px |
| 颜色 | #888888 |
| 背景 | 浅灰色块或虚线框 |

## 二、颜色规范

### 主色调选择

推荐使用单一主色调，保持整体统一：

| 类型 | 推荐色值 |
|------|----------|
| 商务/科技 | #1E88E5（蓝色） |
| 情感/故事 | #D81B60（粉红） |
| 生活/美食 | #43A047（绿色） |
| 教育/知识 | #FB8C00（橙色） |

### 配色原则

1. **全文配色不超过3种**
2. **重点内容用主色**
3. **次要内容用灰色**
4. **避免使用纯黑和纯红**

## 三、间距规范

### 段落间距

| 项目 | 推荐值 |
|------|--------|
| 段间距 | 10-15px |
| 章节间距 | 20-30px |

### 页边距

| 项目 | 推荐值 |
|------|--------|
| 左边距 | 10-15px |
| 右边距 | 10-15px |

### 图片间距

| 项目 | 推荐值 |
|------|--------|
| 图片与正文 | 10-15px |
| 图片之间 | 5-10px |

## 四、段落规范

### 段落长度

- **每段不超过4-5行**
- **每段不超过150字**
- 长段落拆分为多个短段落

### 段落开头

- 正文段落：不需要缩进，顶格写
- 引用段落：可以缩进，或加背景色

### 空行使用

- 段落之间：空1行
- 章节之间：空2行
- 图片前后：空1行

## 五、图片规范

### 图片尺寸

| 类型 | 推荐尺寸 |
|------|----------|
| 正文图片 | 宽度不超过600px |
| 封面图 | 900×383px 或 900×500px |
| 头图 | 900×383px |

### 图片质量

- 使用清晰的图片
- 压缩后上传，保证加载速度
- 尽量使用横图，避免竖图占满屏幕

### 图片来源

- 使用正版图片
- 自己拍摄或设计
- 注明图片来源

## 六、排版工具推荐

### 常用工具

| 工具 | 特点 |
|------|------|
| 秀米 | 模板丰富，适合新手 |
| 135编辑器 | 功能强大，专业排版 |
| i排版 | 界面简洁，操作便捷 |

### 排版流程

1. 先在编辑器中完成排版
2. 复制到公众号后台
3. 再次检查格式
4. 手机预览确认效果

## 七、常见排版错误

### 错误1：颜色过多
- 问题：全文颜色超过5种
- 解决：统一配色，只用主色和灰色

### 错误2：字体过小
- 问题：正文小于14px
- 解决：正文用15-17px

### 错误3：行距过密
- 问题：行距小于1.5
- 解决：行距设置为1.5-1.75

### 错误4：段落过长
- 问题：单段超过200字
- 解决：拆分成多个短段落

### 错误5：图片过多
- 问题：图片超过正文比例
- 解决：精简图片，突出重点

## 八、排版检查清单

发布前确认以下项目：

- [ ] 正文字号15-17px
- [ ] 正文颜色为深灰
- [ ] 行距1.5-1.75
- [ ] 段间距适当
- [ ] 全文配色不超过3种
- [ ] 每段不超过5行
- [ ] 图片清晰且尺寸合适
- [ ] 手机预览效果良好
