# 发布检查清单

> 发布前的最后检查，避免常见错误

## 一、内容检查

### 标题检查
- [ ] 标题简洁有力（不超过20字）
- [ ] 标题不含敏感词
- [ ] 标题与内容一致

### 正文检查
- [ ] 无错别字和语法错误
- [ ] 数据来源已标注
- [ ] 人名地名拼写正确
- [ ] 引用内容注明出处

### 事实核查
- [ ] 数据准确性已核实
- [ ] 案例真实性已确认
- [ ] 新闻时效性已检查

## 二、图片检查

### 封面图
- [ ] 尺寸正确（900×383px 或 900×500px）
- [ ] 图片清晰无水印
- [ ] 与文章内容相关
- [ ] 不含敏感内容

### 正文图片
- [ ] 图片数量适中（不超过10张）
- [ ] 每张图片有说明
- [ ] 图片版权合规
- [ ] 图片加载测试正常

## 三、排版检查

### 格式检查
- [ ] 字号、行距一致
- [ ] 颜色不超过3种
- [ ] 段间距适当
- [ ] 标题层级清晰

### 显示检查
- [ ] 手机预览效果正常
- [ ] 不同手机型号测试
- [ ] 图片显示正常
- [ ] 长按复制测试

## 四、合规检查

### 敏感词检查
- [ ] 无政治敏感内容
- [ ] 无违规广告用语
- [ ] 无虚假宣传内容
- [ ] 无侵权内容

### 版权检查
- [ ] 图片来源合法
- [ ] 文字引用注明出处
- [ ] 转载已获授权

## 五、功能检查

### 链接检查
- [ ] 外链跳转正常
- [ ] 阅读原文链接正确
- [ ] 二维码可扫描

### 互动设置
- [ ] 评论设置正确
- [ ] 原文链接设置
- [ ] 封面摘要填写

## 六、发布信息

### 摘要填写
- [ ] 摘要简洁（不超过50字）
- [ ] 摘要与标题呼应
- [ ] 摘要吸引点击

### 分类设置
- [ ] 文章分类正确
- [ ] 标签设置合理

## 七、发布后检查

- [ ] 手机端阅读体验
- [ ] 分享功能测试
- [ ] 评论功能测试
- [ ] 数据正常显示

## 八、常见问题速查表

| 问题 | 解决方案 |
|------|----------|
| 图片不显示 | 检查图片格式和大小 |
| 排版错乱 | 手机预览后调整 |
| 链接失效 | 检查链接格式 |
| 评论不显示 | 检查评论设置 |
| 分享卡片空白 | 检查封面和摘要 |
