# 结构模板库

> 10种常见文章结构，快速搭建内容框架

## 一、总分总结构

最经典的结构，适合观点类文章。

### 结构示意
```
开头：抛出核心观点（总）
中间：分论点1 + 分论点2 + 分论点3（分）
结尾：重申观点，升华主题（总）
```

### 适用场景
- 观点类文章
- 干货类文章
- 清单类文章

### 模板示例
```
标题：[核心观点]

开头（100-150字）：
- 引入话题
- 抛出核心观点

分论点1（300-400字）：
- 小标题
- 核心论述
- 案例/数据支撑

分论点2（300-400字）：
- 小标题
- 核心论述
- 案例/数据支撑

分论点3（300-400字）：
- 小标题
- 核心论述
- 案例/数据支撑

结尾（100-150字）：
- 重申核心观点
- 升华/呼吁
```

## 二、问题解决结构

适合干货类、教程类文章。

### 结构示意
```
开头：提出问题
中间：分析原因 + 提供解决方案
结尾：总结要点 + 行动建议
```

### 模板示例
```
标题：[问题]+解决方案

开头（100字）：
- 描述问题现状
- 引发共鸣

问题分析（200字）：
- 为什么会出现这个问题
- 不解决的后果

解决方案1（300字）：
- 方法描述
- 具体步骤
- 注意事项

解决方案2（300字）：
- 方法描述
- 具体步骤
- 注意事项

解决方案3（300字）：
- 方法描述
- 具体步骤
- 注意事项

结尾（100字）：
- 总结要点
- 行动建议
```

## 三、故事结构

适合人物故事、案例分享类文章。

### 结构示意
```
开头：设置悬念
发展：冲突出现
高潮：问题解决
结局：收获与感悟
```

### 模板示例
```
标题：[故事悬念]

开头（100字）：
- 人物介绍
- 设置悬念

困境（300字）：
- 遇到的问题/挑战
- 内心挣扎

转折（300字）：
- 关键转折点
- 做出的选择

行动（300字）：
- 具体行动
- 克服困难

收获（200字）：
- 最终结果
- 人生感悟

结尾（100字）：
- 读者启示
- 互动引导
```

## 四、对比结构

适合对比分析、观点辩论类文章。

### 结构示意
```
开头：提出对比主题
中间：A观点 + B观点 + 对比分析
结尾：结论与建议
```

### 模板示例
```
标题：[A]vs[B]，你选哪个？

开头（100字）：
- 引出对比主题
- 说明背景

观点A（300字）：
- 观点阐述
- 支持论据
- 适用场景

观点B（300字）：
- 观点阐述
- 支持论据
- 适用场景

对比分析（300字）：
- 相同点
- 不同点
- 适用人群

结尾（100字）：
- 核心结论
- 选择建议
```

## 五、递进结构

适合深度分析、层层深入类文章。

### 结构示意
```
第一层：表象（是什么）
第二层：原因（为什么）
第三层：本质（底层逻辑）
第四层：方法（怎么做）
```

### 模板示例
```
标题：[主题]的底层逻辑

第一层：现象描述（200字）
- 当前现象
- 数据/案例

第二层：原因分析（300字）
- 表面原因
- 深层原因

第三层：本质探讨（300字）
- 底层逻辑
- 普遍规律

第四层：行动方法（300字）
- 如何应对
- 具体步骤

结尾（100字）：
- 核心观点
- 行动呼吁
```

## 七、问答结构

适合访谈、FAQ类文章。

### 结构示意
```

结尾：总结要点
```

### 模板示例
```

开头（50字）：
- 主题引入
- 为什么重要

Q1：[问题1]
A1：（200字）
- 直接回答
- 延伸解释
- 案例/数据

Q2：[问题2]
A2：（200字）
- 直接回答
- 延伸解释
- 案例/数据

Q3：[问题3]
A3：（200字）
- 直接回答
- 延伸解释
- 案例/数据

结尾（50字）：
- 总结要点
- 引导互动
```

### 结构示意

中间：条目1、条目2、条目3...
结尾：总结推荐
```

### 模板示例

- 清单主题
- 评选标准

条目1：
- 名称
- 推荐理由（100字）
- 适用场景

条目2：
- 名称
- 推荐理由（100字）
- 适用场景

...

结尾（100字）：
- 总结特点
- 使用建议
```

## 九、时间线结构

适合发展历程、成长记录类文章。

### 结构示意
```
开头：时间线背景
中间：阶段1 → 阶段2 → 阶段3 → 阶段4
结尾：现状与展望
```

### 模板示例
```
标题：[主题]发展历程

开头（100字）：
- 时间范围
- 核心变化

阶段1：[时间节点]
- 背景
- 关键事件
- 成果（150字）

阶段2：[时间节点]
- 背景
- 关键事件
- 成果（150字）

阶段3：[时间节点]
- 背景
- 关键事件
- 成果（150字）

结尾（100字）：
- 现状总结
- 未来展望
```

## 十、模块组合结构

适合长文、综合性文章。

### 结构示意
```
开头：主题引入
模块1：[独立主题]
模块2：[独立主题]
模块3：[独立主题]
结尾：整合总结
```

### 模板示例
```
标题：[综合主题]

开头（150字）：
- 主题引入
- 文章结构预告

模块1：[小标题]（400字）
- 独立完整的内容模块

模块2：[小标题]（400字）
- 独立完整的内容模块

模块3：[小标题]（400字）
- 独立完整的内容模块

结尾（150字）：
- 各模块关联
- 核心总结
- 行动建议
```

## 如何选择结构

| 文章类型 | 推荐结构 |
|----------|----------|
| 观点类 | 总分总、递进结构 |
| 干货类 | 问题解决、清单结构 |
| 故事类 | 故事结构、时间线 |
| 对比类 | 对比结构 |
| 访谈类 | 问答结构 |
| 工具推荐 | 清单结构 |
| 案例分析 | 案例结构 |
| 长文 | 模块组合结构 |
