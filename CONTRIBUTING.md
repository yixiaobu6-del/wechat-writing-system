# 贡献指南

感谢你考虑为wechat-writing-system做出贡献！

## 如何贡献

### 报告问题

如果你发现问题，请通过 GitHub Issues 提交，包含：
- 问题的详细描述
- 复现步骤
- 预期行为与实际行为

### 提交代码

1. Fork 本仓库
2. 创建分支 (`git checkout -b feature/your-feature`)
3. 提交更改 (`git commit -m '添加某某功能'`)
4. 推送分支 (`git push origin feature/your-feature`)
5. 创建 Pull Request

### 代码规范

- Python 代码遵循 PEP 8
- 函数添加类型注释和 docstring
- 提交前运行测试确保通过

## 许可证

wechat-writing-system采用 MIT 许可证，贡献的代码将以相同许可证发布。
